## About This Playbook

A governance framework for family offices, private banks, notaries, tax
advisors, and wealth managers to implement structured digital asset
inheritance services — without holding client assets in custody.

This playbook is the written foundation of the DAIO framework. The
interactive companion platform ([daio-unified](../daio-unified/)) mirrors
each chapter as a working toolset.

## Quick Start

- **Days 1–2:** Read [Chapter 1](./ch1-executive-summary.md) and
  [Chapter 3](./ch3-sovereign-guard.md). Skim [Chapter 6](./ch6-tools-templates.md).
- **Days 3–4:** Brief your management, compliance, and operations teams.
- **Days 5–7:** Select a pilot client and prepare execution.
- **Weeks 2–4:** Execute the first Inheritance Container end-to-end.
- **Day 30:** Review, document lessons, scale.
